= Required packages to build SageMath on Ubuntu =

Here is the list of Ubuntu packages to install in order to build SageMath from sources (this is for Ubuntu 16.04):
 * git
 * m4
 * gfortan5
 * libssl-dev
 * python-pip
 * ffmpeg
 * libavdevice-dev
 * pandoc   ''(to convert Jupyter notebooks to pdf)''
